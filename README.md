## 功能概览

| 端 | 说明 |
|----|------|
| **小程序 (app)** | 用户端：发单/接单、订单状态、地址、交易与余额、开屏/轮播/通知/商家推广、个人中心与头像等 |
| **管理后台 (admin)** | 运营与系统管理：订单/用户/学校/订单类型、开屏/滚动通知/首页轮播/商家推广、角色与菜单权限、后台账号、系统参数；HTTPS 开发服务器 |
| **后端 (springboot)** | 统一 API：REST + WebSocket、JWT 鉴权、S3 兼容存储、微信登录与支付、HTTPS（可选） |

- **双身份**：同一账号可同时作为「发单人」与「接单人」使用。  
- **订单类型**：由后台配置（代取快递/外卖、搬运、代购等），支持自定义表单与状态流转。  
- **权限**：后台按角色配置菜单，侧栏与路由按当前用户权限展示与拦截。

## 项目结构

```text
Dongpaopao/
├── app/                    # 微信小程序 (uni-app Vue3)
│   ├── pages/
│   └── uni_modules/
├── admin/                  # 管理后台 (Vue3 + Vite + Element Plus)
│   ├── src/
│   └── vite.config.js
├── springboot/             # 后端 (Spring Boot 3.5, Java 21)
│   └── src/main/
├── sql/
│   └── pao.sql             # MySQL 建表脚本
├── certs/                  # 自签证书 (cert.pem, key.pem)，前后端 HTTPS 可选
└── README.md
```

## 技术栈

| 层级 | 技术 |
|------|------|
| 小程序 | uni-app、Vue 3、uview-plus |

| 后端 | Spring Boot 3.5、MyBatis-Plus、JWT、WebSocket |
| 数据与存储 | MySQL 8、Redis；RustFS对象存储 |
